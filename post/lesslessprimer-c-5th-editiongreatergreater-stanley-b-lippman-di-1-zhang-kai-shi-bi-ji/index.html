编译、运行

C++源文件，常见后辍名：.cc、.cxx、.cpp、.cp、.c。
头文件，常见后辍名：.h、.H、.hpp、.hxx。标准库头文件无后辍。

Windows系统下执行文件不需要后辍名，如$ prog1。Unix系统下需要，如$ a.out。一些系统中必须指名路径(即使在当前目录)。Windows使用反斜杠，如$ .\prog1，Unix使用斜杠，$ ./a.out。

Windows系统中使用$ echo %ERRORLEVEL%，显示上一个执行的程序的main的返回值。Unix中对应的命令是$ echo $?

gcc、g++使用-std=c++0x启动C++11支持。-Wall启用警告。
VS编译命令$ cl /EHsc prog1.cpp，生成prog1.exe。cl为编译器，/EHsc打开标准异常处理，/W4启动警告。

输入、输出

包含四个对象：

  1. istream类型：cin(标准输入)
  2. ostream类型：cout(标准输出)、cerr(标准错误)、clog(用于输出一般信息)

endl：换行，并将输出缓冲刷到设备中。
std::cin >> v1 >> v2;等价于std::cin >> v1; std::cin >> v2;。
while(std::cin >> value)，当遇到EOF，或无效果输入(这里为非整数)时，cin为false。
EOF，Windows快捷键为Ctrl+Z回车，Unix为Ctrl+D。
